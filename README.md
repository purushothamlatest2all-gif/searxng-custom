## ✨ Key Features

- 🤖 **AI Overview** - LexRank-powered extractive summarization
- 🔤 **Spell Checker** - SymSpell real-time correction
- 📊 **Statistics** - Redis-based search analytics
- 👤 **Privacy** - User agent rotation
- 🎨 **Modern UI** - Custom gradient design
